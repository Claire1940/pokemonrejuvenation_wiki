<Section icon={Shield} title="Processo de Aquisição Passo a Passo">

Depois de ter os itens necessários e atingir o ponto correto na história, siga estes passos para garantir seu Type: Null. Este processo envolve uma mistura de exploração, resolução de quebra-cabeças e uma batalha de encontro estático.

<Steps>
  <Step num="1" title="Garanta a Null Key">
    Vá para os **Apartamentos de Grand Dream City** e fale com o cientista na sala mais à direita no segundo andar. Se você completou a missão secundária pré-requisito "Missing Files", ele lhe entregará a **Null Key**.
  </Step>
  <Step num="2" title="Entre no Porão do Hospital">
    Vá ao **Hospital of Hope** no Distrito Residencial. Use o elevador para chegar ao andar B1, depois encontre a escada escondida atrás das máquinas de venda automática para chegar ao nível restrito B2.
  </Step>
  <Step num="3" title="Desbloqueie a Câmara Criogênica">
    Use a Null Key no console central. Você será solicitado a inserir uma senha encontrada nos documentos do "Projeto: Silvally". A senha é frequentemente aleatória com base no seu ID de Sav, então leia as notas próximas com atenção.
  </Step>
  <Step num="4" title="O Encontro Estático">
    O Type: Null estará no **Nível 60**. Ele tem uma taxa de captura alta em comparação com Lendários, mas recomenda-se usar uma **Heavy Ball** ou **Dusk Ball** dado o ambiente de laboratório.
  </Step>
  <Step num="5" title="Calibração Final">
    Após capturá-lo, fale com o NPC próximo para receber o primeiro conjunto de **Memory Drives** (Fogo, Água e Planta).
  </Step>
</Steps>

<Callout type="success" title="Soft Resetting">
Como este é um encontro estático, você pode salvar seu jogo diretamente na frente da câmara criogênica. Isso permite que você faça soft-reset para buscar uma natureza **Jolly** ou **Adamant**, que são ideais para sua evolução, Silvally.
</Callout>

</Section>

<Section icon={Sparkles} title="Evolução e Localização dos Memory Drives">

Para evoluir o Type: Null para Silvally, você deve maximizar sua **Amizade** (Felicidade) e subir um nível. Em Pokémon Rejuvenation, existem várias maneiras de acelerar este processo, o que é vital se você quiser usar o RKS System para combater Líderes de Ginásio específicos.

**Métodos para Aumentar a Amizade:**
- Andar com o Type: Null na sua equipe.
- Usar **Pomeg, Kelpsy ou Qualot Berries** (disponíveis na Loja de Departamentos de GDC).
- Vencer batalhas sem que o Pokémon desmaie.
- Dar a ele um **Soothe Bell**, que pode ser encontrado cedo no jogo perto da Vila Sheridan.

**Localizações dos Memory Drives (Atualização V14):**
| Tipo de Drive | Localização | Capítulo |
| :--- | :--- | :--- |
| **Fogo/Água/Planta** | Recebido junto com o Type: Null | Capítulo 12 |
| **Elétrico/Gelo** | Mercado Subterrâneo de GDC | Capítulo 13 |
| **Aço/Fada** | Recompensa da Missão "The Iron Kingdom" | Capítulo 14 |
| **Dragão/Fantasma** | Escondido nas Ruínas da Zone Zero | Capítulo 15 |

<Callout type="warning" title="Interação com Efeito de Campo">
Lembre-se de que o **Multi-Attack** do Silvally muda de tipo com base no Memory Drive segurado. Em muitos Ginásios de Aevium, o Efeito de Campo pode fortalecer ainda mais este movimento. Por exemplo, um Silvally com Fire Memory usando Multi-Attack em um campo de **Volcanic Topography** causa danos massivos.
</Callout>
