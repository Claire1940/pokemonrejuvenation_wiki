<Section icon={Settings} title="Entendendo o Modo Debug de Pokémon Rejuvenation">

O **modo debug de Pokémon Rejuvenation** é um poderoso conjunto de ferramentas de desenvolvedor e atalhos integrados projetados para ajudar os jogadores a personalizar sua experiência na vasta região de Aevium. Na atualização V14 mais recente, que abrange os Capítulos 1 a 16, a complexidade das rotas da história — especificamente os caminhos Paragon e Renegade — torna as ferramentas de debug quase essenciais para jogadores que desejam explorar todos os ramos narrativos sem repetir mais de 100 horas de conteúdo.

Se você está procurando a versão "Lite" através do sistema de senhas no jogo ou o menu de desenvolvedor completo usado pelos criadores, entender as limitações é vital. A build V14 é um pacote independente que não requer um emulador, o que significa que o ambiente de debug é nativo do motor RPG Maker XP no qual foi construído.

<Callout type="tip" title="Dica de Profissional">
Se você é novo no jogo, comece com o Sistema de Senhas antes de tentar hackear os scripts internos. A maioria dos "cheats" que os jogadores desejam, como dinheiro infinito ou IVs perfeitos, já é oficialmente suportada através de senhas.
</Callout>

<Section icon={Terminal} title="Passo a Passo: Ativando o Menu Debug">

Para aqueles que precisam de mais do que apenas senhas, ativar o menu completo do **modo debug de Pokémon Rejuvenation** requer uma pequena modificação nos arquivos internos do jogo. Este menu concede a você o "Modo Deus" de Aevium, permitindo atravessar paredes, definir seu próprio clima e manipular valores de relacionamento com NPCs como Melia ou Ren.

Siga estes passos cuidadosamente para garantir que você não quebre sua instalação da V14.

<Steps>
  <Step num="1" title="Backup dos seus Arquivos de Save">
    Antes de tocar em qualquer dado do jogo, localize sua pasta de save (geralmente em `C:\Users\[Usuário]\Saved Games\Pokemon Rejuvenation`). Copie a pasta inteira para uma unidade separada. Os saves da V14 não são compatíveis com scripts antigos da V13.5, então garanta que seu backup esteja atualizado.
  </Step>
  <Step num="2" title="Localize o Arquivo de Scripts">
    Navegue até o diretório principal do seu Pokémon Rejuvenation V14. Abra a pasta `Data`. Você está procurando por um arquivo chamado `Scripts.rxdata`. Note que em algumas versões, isso pode estar compactado ou oculto; se for o caso, você pode precisar de um script dedicado "Debug Enabler" fornecido pela comunidade nos fóruns do Reborn Evolved.
  </Step>
  <Step num="3" title="Aplique a Edição de Debug">
    Usando um editor de script ou um arquivo de script de debug pré-modificado, você deve definir a variável `$DEBUG` como `true`. Alternativamente, muitos jogadores usam o mod "Modular Workshop", que adiciona uma opção de Debug alternável diretamente no menu de Opções do jogo sem edição manual de arquivos.
  </Step>
  <Step num="4" title="Verifique no Jogo">
    Inicie o jogo. Abra seu menu principal (onde você vê Pokémon, Bolsa, Salvar). Se for bem-sucedido, uma nova opção "Debug" aparecerá na parte inferior da lista. Você também pode pressionar F9 enquanto estiver no mapa para abrir o menu de Debug Rápido.
  </Step>
</Steps>

<Callout type="warning" title="Aviso: Integridade do Save">
Usar a função "Warp" (Teletransporte) para entrar em áreas antes que a história as acione pode quebrar permanentemente seu arquivo de save. Se você se teletransportar para uma área do Capítulo 16 enquanto estiver no Capítulo 2, o jogo pode falhar ao carregar scripts essenciais de NPCs.
</Callout>

</Section>

<Section icon={Wand2} title="Principais Funções do Menu Debug">

Depois de ativar o **modo debug de Pokémon Rejuvenation** completo, você terá acesso a centenas de comandos. Para a maioria dos jogadores, apenas alguns são realmente necessários para o dia a dia.

### Comandos de Mapa (Overworld)
- **Warp**: Teletransporte para qualquer ID de mapa no jogo. Note que Aevium é dividida em muitos submapas (Grand Dream City possui dezenas).
- **Set Metadata**: Altere a aparência ou o nome do seu personagem instantaneamente.
- **Switches & Variables**: Esta é a ferramenta mais poderosa. Você pode alternar manualmente a "Switch 001" para dizer ao jogo que você já derrotou um Líder de Ginásio específico ou conheceu um NPC específico.

### Gerenciamento de Pokémon
- **Add Pokémon**: Gere qualquer Pokémon da Gen 1 à Gen 9, incluindo Aevian Forms.
- **Fill Pokédex**: Complete instantaneamente suas entradas na dex local e nacional.
- **Modify Party**: Altere níveis, habilidades, movesets e até o status "Shiny" com um único clique.

<Callout type="info" title="Nota sobre Aevian Forms">
Ao usar o debug para adicionar Pokémon, lembre-se de que as Aevian Forms possuem IDs internos únicos. Pesquisar por "Aevian" na barra de busca do debug é a maneira mais rápida de localizá-las.
</Callout>

| Comando | Uso | Recomendado Para |
| :--- | :--- | :--- |
| **Heal Party** | Restauração total instantânea | Longas explorações de masmorras como a Valor Mountain |
| **Give Item** | Adicione qualquer TM, Crest ou Item Chave | Recuperar itens perdíveis em GDC |
| **Toggle Wall** | Atravessar obstáculos | Corrigir glitches de personagem "preso" |
| **Relationship** | Ver/Editar afinidade com NPC | Garantir que você atenda aos requisitos da rota Paragon |

</Section>
