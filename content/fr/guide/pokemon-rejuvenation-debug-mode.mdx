<Section icon={Terminal} title="Étape par étape : Activer le menu Debug">

Pour ceux qui ont besoin de plus que de simples mots de passe, l'activation du menu complet du **mode debug de Pokémon Rejuvenation** nécessite une petite modification des fichiers internes du jeu. Ce menu vous octroie le "Mode Dieu" d'Aevium, vous permettant de traverser les murs, de définir votre propre météo et de manipuler les valeurs de relation avec des PNJ comme Melia ou Ren.

Suivez attentivement ces étapes pour éviter de casser votre installation V14.

<Steps>
  <Step num="1" title="Sauvegardez vos fichiers de sauvegarde">
    Avant de toucher aux données du jeu, localisez votre dossier de sauvegarde (généralement dans `C:\Users\[Utilisateur]\Saved Games\Pokemon Rejuvenation`). Copiez l'intégralité du dossier sur un autre lecteur. Les sauvegardes V14 ne sont pas compatibles avec les anciens scripts V13.5, assurez-vous donc que votre sauvegarde est à jour.
  </Step>
  <Step num="2" title="Localisez le fichier Scripts">
    Naviguez vers votre répertoire principal de Pokémon Rejuvenation V14. Ouvrez le dossier `Data`. Vous recherchez un fichier nommé `Scripts.rxdata`. Notez que dans certaines versions, il peut être regroupé ou caché ; si c'est le cas, vous pourriez avoir besoin d'un script "Debug Enabler" dédié fourni par la communauté sur les forums Reborn Evolved.
  </Step>
  <Step num="3" title="Appliquez la modification Debug">
    À l'aide d'un éditeur de script ou d'un fichier de script debug pré-modifié, vous devez définir la variable `$DEBUG` sur `true`. Alternativement, de nombreux joueurs utilisent le mod "Modular Workshop" qui ajoute une option Debug activable directement dans le menu Options du jeu sans édition manuelle de fichiers.
  </Step>
  <Step num="4" title="Vérifiez en jeu">
    Lancez le jeu. Ouvrez votre menu principal (où vous voyez Pokémon, Sac, Sauvegarder). En cas de succès, une nouvelle option "Debug" apparaîtra en bas de la liste. Vous pouvez également appuyer sur F9 en étant sur la carte pour ouvrir le menu Quick Debug.
  </Step>
</Steps>

<Callout type="warning" title="Attention : Intégrité de la sauvegarde">
L'utilisation de la fonction "Warp" (Téléportation) pour entrer dans des zones avant que l'histoire ne les déclenche peut corrompre définitivement votre fichier de sauvegarde. Si vous vous téléportez dans une zone du Chapitre 16 alors que vous êtes au Chapitre 2, le jeu peut échouer à charger les scripts essentiels des PNJ.
</Callout>

</Section>

<Section icon={Wand2} title="Fonctions clés du menu Debug">

Une fois le **mode debug de Pokémon Rejuvenation** complet activé, vous aurez accès à des centaines de commandes. Pour la plupart des joueurs, seules quelques-unes sont réellement nécessaires au quotidien.

### Commandes d'exploration
- **Warp** : Téléportation vers n'importe quel ID de carte du jeu. Notez qu'Aevium est divisée en de nombreuses sous-cartes (Grand Dream City en possède des dizaines).
- **Set Metadata** : Changez l'apparence ou le nom de votre personnage à la volée.
- **Switches & Variables** : C'est l'outil le plus puissant. Vous pouvez basculer manuellement l'interrupteur "Switch 001" pour dire au jeu que vous avez déjà vaincu un champion d'arène spécifique ou rencontré un PNJ particulier.

### Gestion des Pokémon
- **Add Pokémon** : Générez n'importe quel Pokémon de la Gen 1 à la Gen 9, y compris les Formes d'Aevium.
- **Fill Pokédex** : Complétez instantanément vos entrées de Pokédex local et national.
- **Modify Party** : Changez les niveaux, talents, capacités et même le statut "Shiny" en un clic.

<Callout type="info" title="Note sur les Formes d'Aevium">
Lorsque vous utilisez le debug pour ajouter des Pokémon, n'oubliez pas que les Formes d'Aevium ont des IDs internes uniques. Rechercher "Aevian" dans la barre de recherche du debug est le moyen le plus rapide de les localiser.
</Callout>

| Commande | Utilisation | Recommandé pour |
| :--- | :--- | :--- |
| **Heal Party** | Restauration complète instantanée | Longues explorations de donjons comme le Mont Valor |
| **Give Item** | Ajouter n'importe quelle CT, Crest ou Objet Rare | Récupérer des objets manquables dans GDC |
| **Toggle Wall** | Traverser les obstacles | Réparer les bugs de personnage "coincé" |
| **Relationship** | Voir/Éditer l'affinité PNJ | S'assurer de remplir les conditions de la route Paragon |
