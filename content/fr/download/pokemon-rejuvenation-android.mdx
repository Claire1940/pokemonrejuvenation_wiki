<Section icon={Target} title="Guide d'installation étape par étape">

Configurer **Pokemon Rejuvenation sur Android** demande de la précision. Contrairement aux fichiers APK standards, vous exécutez essentiellement un exécutable Windows via une couche de compatibilité. Suivre la séquence correcte est vital pour éviter que le jeu ne plante pendant la cinématique d'ouverture ou lors du premier combat.

| Étape d'installation | Action requise | Fichier/App clé |
| :--- | :--- | :--- |
| **Étape 1** | Télécharger l'émulateur | APK Joiplay |
| **Étape 2** | Télécharger l'interprète | APK Plugin RPGM |
| **Étape 3** | Obtenir les fichiers du jeu | Zip Pokemon Rejuvenation |
| **Étape 4** | Extraction des fichiers | ZArchiver ou Gestionnaire de fichiers |
| **Étape 5** | Configuration | Paramètres de l'application Joiplay |

<Steps>
  <Step num="1" title="Télécharger et installer les applications de base">
    Tout d'abord, vous devez acquérir les dernières versions de **Joiplay** et du **Plugin RPGM**. Celles-ci sont souvent disponibles sur le Patreon du développeur ou le site officiel. Installez d'abord l'APK Joiplay, suivi immédiatement du Plugin RPGM. N'essayez pas de lancer le jeu sans le plugin, car il fournit les bibliothèques nécessaires pour lire les données du jeu.
  </Step>
  <Step num="2" title="Préparer les fichiers du jeu">
    Téléchargez le dossier du jeu Pokemon Rejuvenation. Il se présente généralement sous format `.zip` ou `.rar`. Utilisez un outil d'extraction fiable comme ZArchiver pour décompresser les fichiers dans un dossier dédié sur votre stockage interne. Évitez d'utiliser directement le dossier "Downloads" ; créez un dossier nommé "Jeux" pour garder des chemins d'accès simples.
  </Step>
  <Step num="3" title="Ajouter le jeu à Joiplay">
    Ouvrez Joiplay et appuyez sur l'icône **"+"**. Sélectionnez "Choisir" et naviguez jusqu'au dossier où vous avez extrait le jeu. Cherchez le fichier `Game.exe`. Appuyez dessus, puis renseignez le nom du jeu et la version (ex: "Rejuvenation v13.5").
  </Step>
  <Step num="4" title="Premier lancement et correctifs">
    Appuyez sur l'icône du jeu dans Joiplay. Lors du premier lancement, l'application peut demander d'extraire un "Runtime Package" (RTP). Laissez ce processus se terminer. Si le jeu demande une mise à jour de script ou un patch, suivez les instructions à l'écran pour assurer la compatibilité avec l'interprète mobile.
  </Step>
</Steps>

<Callout type="warning" title="Exigence Importante">
Accordez toujours les permissions "Accès à tous les fichiers" à Joiplay lorsqu'elles sont demandées. Android 11 et versions ultérieures ont des règles de stockage restreintes strictes ; sans cette permission, Joiplay ne peut pas lire les ressources du jeu, ce qui entraîne un écran noir ou un plantage immédiat.
</Callout>

<Section icon={Shield} title="Optimisation et dépannage">

Exécuter un jeu RPG Maker lourd sur Android peut entraîner des goulots d'étranglement au niveau des performances. Si vous rencontrez des ralentissements, des saccades audio ou des bugs graphiques, il est nécessaire d'ajuster les paramètres de Joiplay. Le matériel mobile gère le "Pathfinding" et le "Traitement des Événements" différemment des PC, une optimisation est donc généralement requise.

| Problème courant | Cause probable | Solution recommandée |
| :--- | :--- | :--- |
| **Écran noir** | Plugin RPGM manquant | Réinstaller la dernière version de l'APK Plugin RPGM |
| **Audio saccadé** | Sous-alimentation du tampon | Désactiver 'Audio de haute qualité' dans les paramètres |
| **Script bloqué** | Version incompatible | Mettre à jour Joiplay vers la dernière version Dev |
| **Texte manquant** | Police non trouvée | Copier le dossier 'Fonts' à la racine de Joiplay |
