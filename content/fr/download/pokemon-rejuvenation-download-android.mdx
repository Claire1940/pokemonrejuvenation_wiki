Pour réussir l'installation de **Pokémon Rejuvenation sur Android**, vous ne pouvez pas simplement installer un seul APK. Le jeu est conçu sur le moteur RPG Maker XP, qu'Android ne prend pas en charge nativement. Vous devez assembler une « suite logicielle » qui permet à votre téléphone d'interpréter la logique du jeu et d'afficher ses graphismes personnalisés.

Assurez-vous toujours que la version de votre plugin RPGM correspond à celle de Joiplay. L'utilisation d'un plugin de 2025 avec une version de Joiplay de 2026 entraîne souvent des erreurs de type « Script Hang » (blocage de script) lors des transitions de combat.
</Callout>

<Section icon={Map} title="Guide d'installation étape par étape">

Suivez attentivement ces étapes pour mener à bien l'installation de **Pokémon Rejuvenation sur Android**. Le non-respect de cet ordre peut entraîner l'échec du jeu lors de la phase d'extraction du package d'exécution.

<Steps>
  <Step num="1" title="Télécharger et installer les APK">
    Téléchargez la dernière version de l'APK Joiplay et de l'APK du plugin RPGM. Installez d'abord Joiplay, puis le plugin. Ne les ouvrez pas encore.
  </Step>
  <Step num="2" title="Préparer les fichiers du jeu">
    Téléchargez le fichier ZIP de Pokémon Rejuvenation. Utilisez une application comme ZArchiver pour extraire le dossier dans le stockage interne de votre téléphone (par exemple, `/Documents/Games/Rejuvenation`).
  </Step>
  <Step num="3" title="Configurer les permissions de Joiplay">
    Ouvrez Joiplay. Acceptez les conditions d'utilisation et accordez l'autorisation « Accès à tous les fichiers ». C'est obligatoire sur Android 11+ pour lire les données du jeu.
  </Step>
  <Step num="4" title="Ajouter le jeu">
    Appuyez sur l'icône **+** dans Joiplay. Sélectionnez **Choisir** (Choose), accédez à votre dossier extrait et sélectionnez le fichier `Game.exe`. Saisissez le nom et la version du jeu comme demandé.
  </Step>
  <Step num="5" title="Configuration du premier lancement">
    Appuyez sur l'icône du jeu. Joiplay vous demandera d'extraire un « Runtime Package ». Laissez ce processus se terminer. Le jeu démarrera ensuite sur le menu principal.
  </Step>
</Steps>

<Callout type="success" title="Succès !">
Si vous voyez l'écran titre avec la musique de la région d'EVM, votre installation est terminée. Vous pouvez maintenant configurer vos touches virtuelles dans la barre latérale de Joiplay.
</Callout>

</Section>

<Section icon={Sparkles} title="Optimisation des performances sur mobile">

Faire tourner un fangame aussi riche que Pokémon Rejuvenation sur mobile peut être exigeant. Pour garantir une expérience fluide à 60 FPS et éviter les plantages lors des combats intenses avec Effets de Terrain, vous devriez ajuster les paramètres suivants dans Joiplay.

| Nom du paramètre | Valeur recommandée | Impact |
| :--- | :--- | :--- |
| **Rendu** | OpenGL ES 3.0 | Meilleure stabilité sur les GPU modernes |
| **VSync** | Désactivé | Réduit la latence d'entrée pour les déplacements |
| **Post-traitement** | Activé (Basique) | Corrige les problèmes de scintillement des sprites |
| **Vitesse du texte** | Rapide (En jeu) | Améliore la vitesse de traitement des scripts |
| **Taille du tas (Heap Size)** | 1024 Mo | Alloue plus de RAM au jeu |

<Callout type="info" title="Conseil d'optimisation">
Si vous rencontrez des ralentissements dans les grandes villes comme Gearen City, essayez de désactiver les « Transitions de carte » (Map Transitions) dans les paramètres individuels du jeu sur Joiplay. Cela réduit la charge lors des déplacements entre différentes zones.
</Callout>

**Stratégies d'optimisation clés :**
- **Vider le cache** : Videz régulièrement le cache de Joiplay si le jeu commence à saccader après de longues sessions de jeu.
- **Applications en arrière-plan** : Fermez toutes les applications gourmandes en RAM comme Chrome ou YouTube avant de lancer le jeu.
- **Économiseur de batterie** : Assurez-vous que le mode « Économiseur de batterie » de votre téléphone est désactivé, car il bride le processeur et ralentit l'exécution des scripts.

</Section>

<Section icon={Settings} title="Dépannage des erreurs courantes sur Android">

Même avec une configuration parfaite pour **télécharger Pokémon Rejuvenation sur Android**, les utilisateurs mobiles peuvent rencontrer des bugs spécifiques dus à la nature de l'émulation. Utilisez le tableau ci-dessous pour identifier et résoudre les problèmes courants.

| Description de l'erreur | Cause courante | Correction recommandée |
| :--- | :--- | :--- |
| **Écran noir au démarrage** | Plugin RPGM manquant | Réinstaller l'APK du plugin RPGM |
| **Le script prend trop de temps** | Bridage du processeur | Activer « Ignorer les erreurs de script » dans Joiplay |
| **Fichier de sauvegarde introuvable** | Problème de permission | Déplacer le dossier du jeu vers `/Android/data/` |
| **Pas de son/audio** | Incompatibilité MIDI | Changer le moteur audio pour « Native » dans Joiplay |
| **Touches non réactives** | Bug de superposition de l'affichage | Réinitialiser le pavé tactile virtuel dans le menu latéral |

<Callout type="warning" title="Attention : Sécurité des données de sauvegarde">
Les mises à jour d'Android peuvent parfois effacer les données des applications. Sauvegardez toujours manuellement votre fichier `Save.rxdata` (situé dans le dossier du jeu) sur un service cloud comme Google Drive ou Dropbox.
</Callout>

</Section>
