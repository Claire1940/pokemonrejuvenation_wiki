<Section icon={Shield} title="Mots de passe pour la difficulté et le mode défi">

Pour ceux qui trouvent le jeu de base trop facile, les **mots de passe Pokémon Rejuvenation** offrent plusieurs façons d'augmenter l'intensité. Ces mots de passe peuvent imposer des règles strictes qui n'étaient auparavant possibles que par des défis auto-imposés basés sur le "système d'honneur". En utilisant ces codes, le moteur du jeu appliquera réellement les règles, comme vous empêcher d'utiliser un Pokémon une fois qu'il a été mis K.O.

| Mot de passe | Type de Défi | Niveau d'Application |
| :--- | :--- | :--- |
| **nuzlocke** | Règles Nuzlocke standard | Élevé (Mort permanente) |
| **monotype** | Restreint l'équipe à un seul type | Modéré (Verrouillage de sélection) |
| **flatstats** | Supprime les EVs et IVs pour tous | Élevé (Stratégie pure) |

<Callout type="success" title="L'Expérience Nuzlocke">
Lorsque vous activez le mot de passe `nuzlocke`, le jeu gère automatiquement la règle de la "première rencontre par zone" et déplace définitivement les Pokémon K.O. vers une boîte spéciale "Morts". C'est le moyen le plus stable de jouer un Nuzlocke dans Pokémon Rejuvenation sans enfreindre les règles par accident.
</Callout>

Au-delà des défis standards, le mot de passe `monotype` est un favori pour les parties communautaires. Lorsqu'on vous le demande, vous sélectionnez un type unique (ex: Feu, Spectre ou Acier), et le jeu s'assurera que votre starter et divers Pokémon offerts sont ajustés pour correspondre à ce type dans la mesure du possible. Cela ajoute une incroyable rejouabilité à la vaste région d'Aevium.

<Section icon={Target} title="Étape par étape : comment saisir les mots de passe">

Saisir les **mots de passe Pokémon Rejuvenation** est un processus simple, mais il ne peut être effectué qu'à des moments précis du jeu. Si vous manquez la fenêtre initiale au début d'une nouvelle sauvegarde, vous devrez trouver une Data Chip pour les modifier plus tard.

<Steps>
  <Step num="1" title="Démarrer une nouvelle partie">
    Commencez un nouveau fichier de sauvegarde et suivez le dialogue d'ouverture avec la mère du protagoniste et la séquence initiale du train.
  </Step>
  <Step num="2" title="L'interaction dans le train">
    Pendant que vous êtes dans le train, vous rencontrerez un PC ou serez sollicité par un PNJ. C'est votre première opportunité d'entrer plusieurs mots de passe.
  </Step>
  <Step num="3" title="Entrer les codes">
    Tapez les mots de passe un par un. Le jeu affichera un message de confirmation pour chaque saisie réussie. Vous pouvez en entrer autant que vous le souhaitez.
  </Step>
  <Step num="4" title="Changements en fin de partie">
    Si vous êtes déjà en cours de partie, obtenez une **Data Chip** (trouvée dans divers endroits cachés ou boutiques). Utilisez la Data Chip sur n'importe quel PC dans un Centre Pokémon pour ouvrir le menu des mots de passe.
  </Step>
</Steps>

<Callout type="info" title="Emplacements des Data Chips">
Les Data Chips sont des objets rares en début de partie. Vous pouvez généralement en trouver une dans le laboratoire de Gearen City ou comme récompense de quêtes secondaires. En 2026, certains PNJ du district judiciaire les échangent également contre de petites quantités d'AP (Points de Succès).
</Callout>

Il est important de noter que si la plupart des mots de passe peuvent être désactivés, les mots de passe de défi comme `nuzlocke` ou `monotype` sont souvent permanents pour ce fichier de sauvegarde afin d'empêcher les joueurs de les désactiver lors de combats de boss difficiles. Vérifiez toujours vos choix avant de cliquer sur "Confirmer" sur le PC.

</Section>

Comme mentionné dans les recherches de la communauté, essayer d'utiliser des mots de passe ou des codes de debug pour vous octroyer 999 Prismes Noirs déclenchera le dialogue "Cheater Edition" avec Mr. Luck. Cela entraînera la perte de tous les prismes et aucune récompense. Le jeu est conçu pour détecter les quantités d'objets "impossibles" pour les objets liés aux quêtes.

Le mot de passe `shinycharm` est controversé. Bien qu'il ne vous donne pas directement l'objet Charme Chroma, il modifie le taux de base de chromatiques pour qu'il soit plus proche des jeux de fans modernes (environ 1/100). C'est parfait pour les joueurs qui apprécient l'esthétique des Pokémon chromatiques mais ne veulent pas passer des semaines à faire des soft-resets pour un starter.
